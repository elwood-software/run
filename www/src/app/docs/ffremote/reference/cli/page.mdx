# FFremote CLI

The FFremote CLI (Command Line Interface) allows you to run FFMpeg jobs on a remote server. The CLI has a
number of commands that can be used to perform different tasks, check the
links below for more information on each command.

Visit the [Getting Started Guide](/docs/ffremote/start#install) to learn how to install CLI.

## Commands

- [ffremote execute](#execute) - Execute an FFmpeg on the remote server
- [ffremote list](#list) - List all executions
- [ffremote watch](#watch) - Watch the execution of a command
- [ffremote get](#get) - Download the output of a command
- [ffremote status](#status) - Get the status of a command
- [ffremote auth](#auth) - Authenticate with the remote server

### execute

```bash
# Usage
ffremote execute [options] -- <ffmpeg-args>...
ffremote [options] -- <ffmpeg-args>...
ffremote <ffmpeg-args>...

# Examples
ffremote execute -i "test.mov" -t 30 -c:v libx264 -c:a aac -strict experimental "test.mp4"
ffremote --size=lg -- -i "test.mov" -t 30 -c:v libx264 -c:a aac -strict experimental "test.mp4"
ffremote -i "test.mov" -t 30 -c:v libx264 -c:a aac -strict experimental "test.mp4"
```

#### Options

##### `--size=<size>`

Size of the remote instance to use. Possible values are `sm`, `md`, `lg`, `xl`.
