# FFremote CLI

The FFremote CLI (Command Line Interface) allows you to run FFMpeg jobs on a remote server. The CLI has a
number of commands that can be used to perform different tasks, check the
links below for more information on each command.

## Commands

- [ffr execute](#execute) - Execute an FFmpeg on the remote server
- [ffr list](#list) - List all executions
- [ffr watch](#watch) - Watch the execution of a command
- [ffr get](#get) - Download the output of a command
- [ffr status](#status) - Get the status of a command
- [ffr auth](#auth) - Authenticate with the remote server

### execute

```bash
# Usage
ffr execute [options] -- <ffmpeg-args>...
ffr [options] -- <ffmpeg-args>...
ffr <ffmpeg-args>...

# Examples
ffr execute -i "test.mov" -t 30 -c:v libx264 -c:a aac -strict experimental "test.mp4"
ffr --size=lg -- -i "test.mov" -t 30 -c:v libx264 -c:a aac -strict experimental "test.mp4"
ffr -i "test.mov" -t 30 -c:v libx264 -c:a aac -strict experimental "test.mp4"
```

#### Options

##### `--size=<size>`

Size of the remote instance to use. Possible values are `sm`, `md`, `lg`, `xl`.
